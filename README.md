# task_app
This project is a task manager application written in C++.

Functionnalities :
<pre>
Create a todo item
Update a todo item
Delete a todo item
Set done or undone a todo item
Add a detail for a todo item
Update a detail for todo item
Delete a detail for a todo item
</pre>
